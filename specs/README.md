# Spec-Driven Development

The JSON companion is machine-validation source; Markdown is a Git-review artifact generated or synchronized from it. SDD-2 will add registry traceability without allowing silent disagreement.

`active/` stores non-terminal specs; `completed/` released ones; `deferred/` terminal deferred/rejected/superseded records; `adr/` decisions; `changes/` release records; `contract/` the shared versioned contract.
