# Release documentation

Each YouTrader release is orchestrated by the **AI Release Manager** (CEO interface).

```
Prepare Release X.Y.Z
```

Full architecture: [AI_DEVELOPMENT_WORKFLOW.md](../AI_DEVELOPMENT_WORKFLOW.md)

Bootstrap:

```bash
~/Projects/ai-dev-tools/scripts/prepare-youtrader-release.sh X.Y.Z
```

Output folder:

```
docs/releases/X.Y.Z/
```
